HappeND
=======

A Cordova-based app that scrapes the calendar data off of calendar.nd.edu and displays it on map together with your location via Google Maps. Which day to view events for (±six months) can be selected with the sliding bar at the bottom.

This was my entry for the 2013 AT&T Hackathon at Notre Dame. It won the "best Notre Dame-themed app" prize, which was nice.
